Points and objects trace

When icon is activated, points marked for trace leave a trace when moving.

Some macros also leave traces of objects when executed. When trace mode is activated, these traces remain on the figure.

To get a point marked for trace, use menu item Modify - Mark a point for trace (icon ).

To get a point unmarked for trace, use menu item Modify - Unmark a point for trace (icon ).
