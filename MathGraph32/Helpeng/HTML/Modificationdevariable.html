Variable modification macro

Macros ›› Different kind of macros ››

A macro modifying a variable is created through menu Macros - New macro - Modifying variable.

It's purpose is to modify all the variable attributes and recalculate the figure.


You have to specify the new values of :

  • mini value of the variable
  • maxi value of the variable
  • step value of the variable
  • current value of the variable
